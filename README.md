Inblox AI ✉️

AI Email Writing Assistant (Demo)

Inblox AI is a web-based email writing assistant that helps users generate professional emails by providing a topic or short description.

Users simply enter the subject or purpose of an email, and the system generates a structured email draft automatically.

This project demonstrates the concept of an AI-powered email generator and is created for learning and portfolio purposes.

---

Features

- ✉️ Generate emails from a topic
- ⚡ Instant email draft creation
- 🧠 AI-style writing simulation
- 📱 Mobile-friendly interface
- 🎨 Clean and minimal design

---

How It Works

1. User enters a topic or email purpose
2. The system processes the input
3. A structured email draft is generated automatically

Tech Stack
